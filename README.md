# 豆包网页版
## 要点
1. 一个基于bubblewrap构建的pwa项目
2. 输出在android上运行豆包网页版的apk

## 不就是apk包了一个网页吗？为什么搞这个事？为什么不用豆包官方app？
- 因为官方的移动端app上 没有pdf文档 保留格式的全文翻译，没法截图提问。而这些功能我十分需要在平板上使用。

## 未来迭代完善方向
- 加上启动时从上次退出app时 最近使用的豆包会话启动，不用每次再去列表里找了。
- 加些好的缓存策略
- 优化改动页面排布，更适合移动端的逻辑
- 加按钮，当跳转浏览到外部网页时，可以一键回到原豆包对话，不用再一步步回退。

## 初始备注
1. 目前只是想跑通
2. 没有缓存
3. 没有保持登录
4. 没有启动记住页面
